Ai Powered Photo
Privacy in Social Media

Project Overview

AI-Powered Photo Privacy in Social Media is a privacy-focused web application that enhances the way photos are shared on social platforms. Built using Flask, MySQL, and AI-based face recognition, this project allows users to maintain control over their appearance in photos posted by others.

When a user uploads an image as part of a post, the system automatically uses face detection and recognition to identify individuals in the photo by comparing them with previously trained facial data. If any recognized users are found in the image, they are notified and asked for approval before the post is made public. This ensures that no one’s photo is shared without their explicit consent.

The system includes:

  • User Registration/Login
  • Face Training (via webcam or manual upload)
  • Post Creation with image upload
  • AI-Based Face Detection & Matching
  • Approval/Notification System
  • Admin Panel for user and post management

Technologies Used

Frontend:

  • HTML5, CSS 3
  • JavaScript for interactivity
  • Bootstrap for responsive design

Backend:

  • Python (Flask Framework)
  • Face detection and recognition using face_recognition and Dlib
  • MySQL for database management

Libraries & Tools:

  • OpenCV for image handling and face cropping
  • Secure user authentication using Flask sessions
  • Jinja2 templating engine
  • AJAX and JavaScript for dynamic content updates

Security & Privacy:

  • Face-based post approval system
  • Notification-based share approval
  • Screenshot prevention logic

Key Features

Features includes secure user authentication, enabling users to manage their profiles, update personal information, and upload profile pictures. Users can create image and text posts, which are automatically scanned for faces using AI-based face detection. Recognized faces are matched against trained data, and the system sends approval notifications to the identified users. Posts are only published publicly after receiving consent from all detected users. Users can train their faces via manual image uploads or live webcam capture. The platform includes robust notification and approval systems for both face detection and post sharing, ensuring that any content sharing requires the original poster’s approval. Additionally, screenshot and screen recording protection is implemented to prevent unauthorized capture of content. An admin panel allows management of users, posts, and face data. Built with Flask, MySQL, Bootstrap, and AI libraries like face_recognition, the system delivers a secure, user-controlled, and privacy-first social media experience.
